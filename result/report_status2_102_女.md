# 健康评估与照护行动计划

## 0. 报告说明
本报告基于您/家属提供的信息进行风险提示与照护建议，不能替代医生面诊；涉及用药与检查，请以医生意见为准。

## 1. 健康报告总结
1） 老人家整体身体功能比较弱，目前需要家人全面的帮助才能完成日常生活。
2） 当前最核心的问题是身体活动能力严重下降，同时体重偏轻，营养可能跟不上。
3） 我们建议立刻从“防摔倒、防压疮、防呛咳”这三件最要紧的事做起，同时逐步增加营养和被动活动，让老人家生活得更安全、更舒适。

## 2. 您的健康画像（现在“好在哪里、短板在哪里”）
### （1）功能状态：需要全面照护
老人家因为身体原因，活动非常不方便，洗澡需要家人完全帮忙，出门、做家务、走路、提东西这些事自己都做不了。

### （2）优势（需要继续保持）
*   **头脑清楚**：时间、地点都认得清，算数也没问题，脑子很灵光。
*   **心情平稳**：很少觉得难过、担心或者孤单，情绪一直不错。
*   **没有“老毛病”**：像高血压、糖尿病、心脏病这些常见的老年病都没有，底子很好。
*   **家庭支持好**：和老伴、家人住在一起，有儿媳专门照顾，家里条件也好，还有医保。
*   **生活习惯好**：不抽烟也不喝酒。

### （3）主要问题（本报告重点要解决的）
1.  **身体活动能力严重下降**：这是当前最核心的问题，导致洗澡、吃饭、出门等所有事情都需要依赖家人，也容易引起摔倒、长褥疮等风险。
2.  **体重太轻，营养跟不上**：老人家太瘦了，这会让身体没力气、抵抗力变差，伤口也不容易好，是恢复身体功能的一大障碍。
3.  **完全缺乏活动**：因为动不了，也没有安排任何活动，这会让肌肉越来越没力气，关节变僵硬，身体功能会越来越差。

## 3. 风险因素（按时间或优先级最高的事来写，方便落地）
### 近期（1-4周）重点风险：
*   **摔倒**：晚上起夜上厕所、从床上挪到椅子上、或者在屋里稍微动一下的时候，因为腿没力气、站不稳，很容易摔倒。
*   **长褥疮**：长时间躺在床上或者坐着，屁股、脚后跟这些骨头突出的地方一直压着，皮肤容易缺血、破皮，形成褥疮。
*   **吃东西呛到肺里**：吃饭喝水的时候，如果姿势不对或者吃得急，食物或水可能呛到气管里，引起严重的肺炎。

### 中期（1-6月）重点风险：
*   **肌肉越来越没力气**：一直不动，肌肉会萎缩得更厉害，关节会变僵，最后可能完全下不了床，感染的风险也会增加。
*   **营养越来越差，容易生病**：年纪大了，胃口和消化能力本来就弱，再加上动得少，营养跟不上，身体会越来越瘦弱，更容易得肺炎、尿路感染等病。
*   **变得孤单、反应变慢**：因为出不了门，接触的人和事少了，可能会变得不爱说话、反应迟钝，对大脑的锻炼也不够。

**注：** 风险提示不代表一定会发生，而是提醒优先把“最要命、最可防”的环节补上。

## 4. 健康建议

### A. 第一优先级（立刻要做的事）
**1）打造一个“防摔倒”的安全屋**
- **负责人**：主要照护者（儿媳）及其他家人
- **怎么做**：
    *   在床边放一个便盆或尿壶，减少夜间去厕所的需要。
    *   使用带护栏的护理床，防止睡觉时滚落。
    *   检查家里地面，确保平整、干燥，必要时铺上防滑垫。
    *   老人家要移动时（如起床、坐椅子），一定要有人在旁边扶着或抱着，千万不要让她自己尝试。
- **完成标准**：夜间床边有便器可用；床有护栏；地面无湿滑；每次移动都有家人辅助。

**2）建立“防压疮”翻身时钟**
- **负责人**：主要照护者（儿媳）
- **怎么做**：
    *   定好闹钟，无论白天黑夜，至少每2小时帮老人家翻一次身，变换一下躺或坐的姿势。
    *   在床垫和坐垫上铺上专门的防褥疮气垫。
    *   每天检查屁股、脚后跟、胳膊肘等骨头突出的地方，看皮肤有没有发红、破皮，保持这些地方清洁干燥。
- **完成标准**：翻身时间表已执行；使用了防压疮垫；皮肤完好，没有发红或破损。

**3）学会“安全喂饭”法**
- **负责人**：负责喂饭的家人
- **怎么做**：
    *   喂饭时，一定要把床头摇高，或者用枕头垫高后背，让老人家半坐着吃，绝对不能躺着吃。
    *   喂得慢一点，一口一口来，确认咽下去了再喂下一口。
    *   把食物做得软烂一些，比如做成糊状、粥状，更容易吞咽。
    *   吃完饭后，继续保持半坐姿势至少30分钟，不要马上躺下。
- **完成标准**：每次进食均为半卧位；小口慢喂；食物性状合适；餐后保持坐姿半小时。

### B. 第二优先级（1周内开始，并长期坚持）
**1）每天做“被动健身操”**
- **负责人**：主要照护者（儿媳）
- **怎么做**：
    *   每天固定时间（如上午、下午各一次），家人轻柔地帮老人家活动胳膊、腿的各个关节，做弯曲、伸展的动作。
    *   动作要慢、要轻，以老人家不感到疼痛为准。
    *   如果可能，可以在活动时轻轻施加一点阻力（比如用手托着腿让她稍微用点力蹬），锻炼肌肉力量。
- **完成标准**：每天完成两次全身关节的被动活动；老人家感觉舒适，无疼痛。

**2）实施“增重营养计划”**
- **负责人**：负责做饭的家人
- **怎么做**：
    *   每周固定时间给老人家称一次体重，记录下来。
    *   在饭菜里多加入一些优质蛋白，比如把鱼肉、鸡肉、豆腐、鸡蛋做得特别烂，混在粥、面条里。
    *   采用“少食多餐”的方式，在三餐之外，上午、下午和睡前增加一些营养点心，比如营养奶昔、酸奶、芝麻糊等。
    *   如果吃饭量实在上不去，可以咨询医生或营养师，使用一些肠内营养补充剂（营养粉）。
- **完成标准**：每周记录体重；每餐有高蛋白软食；每日有加餐；体重停止下降或有所增加。

### C. 第三优先级（在A/B级措施稳定后开展）
**1）启动“每日开心一刻”脑力活动**
- **负责人**：所有家人
- **怎么做**：
    *   每天抽时间和老人家聊聊天，说说今天是几号、星期几、什么季节，聊聊家里的亲人。
    *   给老人家听听她年轻时喜欢的戏曲、老歌，或者广播节目。
    *   拿出老照片一起看看，问问“这是谁呀？”“这是在哪儿照的？”，或者拿一些日常物品让她认一认。
- **完成标准**：每天至少有15-20分钟的互动交流；老人家愿意参与，情绪愉悦。
